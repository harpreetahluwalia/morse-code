FUNCTIONALITY OF THE PROGRAM : 

This program implements a morse code encoder/decoder in the C++. In this, user provides a string input and 
it converts the user input to the equivalent morse code and prints the code. 
It also decode the same morse code to its respective alphabets and prints the decoded message as well.

**************************************************************************************************************************

**************************************************************************************************************************
FILES OF THE PROGRAM (INCLUDING HEADERS AND IMPLEMENTATION) : 

tree.cpp	It includes implementation of tree class and deals with inserting node, encoding and decoding message
tree.h		It includes declaration of tree class and deals with inserting node, encoding and decoding message
main.cpp	It is the main program file. It drives the whole application and interacts with the user inputs
morsecode.txt	It includes morse code with respective alphabet
